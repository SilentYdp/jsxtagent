# 江苏信托智能体平台 · React 开发计划

## 项目目标
基于 `docs/agentv2.html` 原型，使用现代 React 技术栈构建江苏信托智能体平台。

## 技术栈
- **框架**: Next.js 14 (App Router) + React 18 + TypeScript
- **样式**: Tailwind CSS
- **状态管理**: Zustand
- **数据获取**: TanStack Query
- **图表**: Chart.js + react-chartjs-2
